# Dixie Flatline Brand

## Positioning

Dixie Flatline is a repo-local memory layer for AI coding assistants.

Tagline:

```txt
Persistent project memory for AI coding agents.
```

One-line pitch:

```txt
Dixie Flatline helps your AI coding assistant remember how your project actually works.
```

Slightly fun version:

```txt
Stop explaining the same repo to your AI every morning.
```

## Tone

- Retro cyberpunk
- Useful, not gimmicky
- Slightly weird
- Developer-first
- Dry humour, not novelty branding

## Visual Metaphor

Stored digital mind, memory ghost, terminal archive.

Avoid horror imagery and avoid leaning too directly on any specific source reference.

## Usage Notes

- Use the mark without tiny text at icon sizes.
- Keep social copy short: `Dixie Flatline` and `Persistent project memory for AI coding agents`.
- Keep README copy practical and developer-first.
- Do not overuse exact pop-culture references. The product identity should stand on its own.

## Regenerating Assets

```bash
pnpm brand
```
