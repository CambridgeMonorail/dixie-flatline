# Roadmap

Dixie Flatline is experimental and pre-1.0. This roadmap is intentionally small.

## Phase 1: Structured Local Memory

- Typed Markdown entries
- Importance, confidence, freshness, and supersession metadata
- Deterministic retrieval ranking

## Phase 2: VS Code Chat Participant

- `@project-memory` chat participant
- Active-file context retrieval
- Cited responses from memory

## Phase 3: Copilot Tool Integration

- Relevant memory tool
- Critical decisions tool
- Conflict and open-question tools
- Instruction compilation

## Phase 4: Diff-Driven Updates

- Diff classification
- Suggested typed memory actions
- Human-confirmed decision recording

## Later

- Extension-host test suite
- Optional embeddings
- MCP server mode
- Multi-repo memory
